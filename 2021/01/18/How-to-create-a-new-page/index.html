How to create a new page

发表于 2021-01-18 阅读次数：
本文字数： 162 阅读时长 ≈ 1 分钟

首先

"hexo new post "name" "

然后目录下输入

"hexo g"

生成静态网页，然后输入

"hexo s"

可以本地预览效果，最后输入

"hexo d"

上传到github上。这时打开你的github.io主页就能看到发布的文章啦。
